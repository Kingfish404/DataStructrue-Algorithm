## HuffmanPicture

用Huffman Tree实现了一个文件的压缩与解压程序 

### 说明

压缩后文件类型为.huf  
格式：  
HEAD文件+Huffman压缩后的内容

解压后在原文件名前加上"Re"

## GraphCPro

基于命令行的景区景点路径规划系统

### 说明

寻路方式为DFS,深度优先遍历

最短路径算法为:Dijkstra算法

最小生成树算法:Prim算法
